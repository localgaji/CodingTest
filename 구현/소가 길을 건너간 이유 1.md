## 소가 길을 건너간 이유 1

https://www.acmicpc.net/problem/14467

---

#### 풀이 1. 

```python
n = int(input())

cow = [None] * 11
answer = 0
for i in range(n):
    x, y = map(int, input().split())
    if cow[x] == None:
        cow[x] = y
    elif cow[x] != y:
        answer += 1
        cow[x] = y
print(answer)
```
