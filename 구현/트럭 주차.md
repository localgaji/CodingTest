#### 풀이 1. 

```python
a,b,c = map(int, input().split())
fees = [a, b, c]
fee = 0
times = [0] * 101
for i in range(3):
    inn, out = map(int, input().split())
    for j in range(inn, out):
        times[j] += 1
for x in times:
    if x != 0:
        fee += x * fees[x-1]
print(fee)
```
